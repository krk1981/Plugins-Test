**Confluence** (for publishing):
When the task mentions Confluence, publishing, or creating pages, you MUST use these tools
to actually publish content to Confluence. Do NOT just create local "Confluence-ready" files.
1. Call `mcp__confluence__confluence_get_spaces` to list spaces and get a `space_id`
2. Use `mcp__confluence__confluence_search_pages` with CQL to find existing pages
3. Use `mcp__confluence__confluence_create_page` to create pages (params: space_id, title, body as markdown, parent_id)
4. Use `mcp__confluence__confluence_update_page` to update existing pages (params: page_id, title, body, version_number)
5. Use `mcp__confluence__confluence_get_page` to get a page by ID (e.g. to get the current version number before updating)
